# Python CLI To-Do App

A simple command-line To-Do application built with Python.  
Users can add tasks, view tasks, delete tasks, and quit the app. The project includes input validation and error handling.

## Features
- Add tasks
- View tasks
- Delete tasks by number
- Input validation for menu choices and deletion
- Error handling for invalid input and empty task list cases

## How to Run
1. Clone the repo
2. Run the script:

```bash
python todo.py
